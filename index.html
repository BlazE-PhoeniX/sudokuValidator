Instructions

Using Voice Transcription

  • Turn on the mic.
  • Allow permission for mic to be used.
  • when you see the splash effect around the mic button, Read the numbers one by one row-wise clearly without breaks.
  • After reading the entire set of numbers, turn off the mic.
  • Check if the numbers are placed in the respective squares, if not try again.

Note:

Voice transcription feature is in beta, hence the numbers can be filled only from start, So if something is read wrongly,
Do not stop, Continue the process, wrongly filled or unfilled numbers can be edited manually.

Manual Typing

  • Zoom in on the Square.
  • Select the square to be edited or filled in with number.
  • Type the correct number.
  • PC users, Use TAB key to move through the squares swiftly.
